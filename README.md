#  Portfolio Website

## Description  
This is the personal portfolio website for **Priya G**, showcasing her profile, skills, projects and contact information. The site is built using a Bootstrap-based template and includes sections such as Home, About, Resume, Projects and Workshops.

## Features  
- Hero section with photo and introduction.  
- Navigation menu for Home, About, Resume, Projects, Workshop and Contact pages.  
- Responsive layout using Bootstrap for mobile and desktop.  
- Social links (GitHub, LinkedIn, Instagram, Twitter) integrated in header and footer.  
- Clean, modern design theme suited for a personal portfolio.  
- Uses AOS animations, swiper, lightbox and other vendor scripts for enhanced UX.  

## Technologies Used  
- HTML5 / CSS3 / JavaScript  
- [Bootstrap](https://getbootstrap.com) (CSS & JS)  
- Vendor libraries: Bootstrap Icons, AOS, Swiper, GLightbox, Isotope, PureCounter  
- Google Fonts: Roboto, Poppins, Raleway  
- A lightweight static website (no backend required)
